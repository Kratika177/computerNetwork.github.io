What is Computer Networks?

1- Computer Network is a collection of computer devices.
2- The purpose of computer network is to share the data between computers.

3-The data sent by the sender must be understand by receiver to assure this there must be protocol.
4- To Communicate between client and server present in different country smoothly we need Computer Networks.
